# Welcome to Sistema de Abogados

Sistema de Abogados is a comprehensive case management system designed specifically for law firms to streamline their operations. Built with Laravel 9, this system provides a complete solution for managing clients, legal cases, conciliation processes, documents, and activities.

## Technology Stack

Sistema de Abogados is built with modern, reliable technologies:

- **Laravel 9** - Robust PHP framework for backend operations
- **Laravel Breeze** - Authentication scaffolding with Blade templates
- **Spatie Laravel Permission** - Role and permission management
- **Laravel Scout** - Full-text search capabilities
- **TailwindCSS** - Modern utility-first CSS framework
- **Alpine.js** - Lightweight JavaScript framework for interactivity
- **FullCalendar.js** - Interactive calendar and scheduling
- **Chart.js** - Dashboard analytics and visualizations
- **SweetAlert2** - Beautiful alert modals
- **MySQL** - Reliable relational database

<Note>
This system is designed for law firms that need to manage both traditional legal cases (Casos) and conciliation processes (Expedientes) in a unified platform.
</Note>
